# GoCoding — Roblox Coding & 3D Modeling Landing (NYC)

Конверсионный лендинг для GoCoding: курсы программирования и 3D‑моделирования в Roblox для детей в Нью‑Йорке. Одностраничный сайт с записью на пробное занятие за $20.

> **Self‑contained.** Весь сайт — это один файл `index.html`. Все изображения и логотип встроены прямо в файл (base64), внешних зависимостей нет. Страница открывается и работает где угодно — локально, на GitHub Pages, на любом статическом хостинге, даже офлайн.

---

## ✨ Что внутри

- **Адаптивный дизайн** — выверенная десктоп‑ и мобильная версии (проверены до 360px), без горизонтального скролла и наложений.
- **Мобильное меню** — выпадающая навигация с анимацией бургера в крестик.
- **Анимации и интерактив:**
  - залипающая шапка с тенью при скролле;
  - плавный скролл по якорям;
  - аккордеон FAQ;
  - «дышащий» виджет записи и плавающие бейджи;
  - анимация дефицита (прогресс‑бар + счётчик мест 26/30);
  - всплывающее окно через 25 секунд (с защитой от повторного показа);
  - reveal‑анимации секций при скролле;
  - форма → лоадер → экран «Спасибо» + событие `Lead` в Meta Pixel.
- **Цветовой ритм секций** — мягкие брендовые оттенки (голубой / тёплый / лавандовый) с тёмными CTA‑блоками.
- **Сравнительная таблица** — на мобильном перестраивается в аккуратные карточки.

## 🚀 Локальный запуск

Достаточно открыть `index.html` в браузере двойным кликом. Либо поднять локальный сервер:

```bash
# Python
python3 -m http.server 8000
# затем откройте http://localhost:8000

# или Node
npx serve .
```

## 🌐 Деплой на GitHub Pages

1. Запушьте репозиторий на GitHub (см. ниже).
2. В репозитории: **Settings → Pages**.
3. **Source:** `Deploy from a branch` → ветка `main`, папка `/ (root)` → **Save**.
4. Через ~1 минуту сайт будет доступен по адресу:
   `https://<ваш‑логин>.github.io/gocoding-landing/`

Свой домен: добавьте файл `CNAME` с доменом в корень и пропишите DNS у регистратора.

## 🛠 Как редактировать

Всё в `index.html`:
- **Стили** — в `<style>` в `<head>` (CSS‑переменные с токенами в `:root` — цвета, радиусы, тени).
- **Контент** — в `<body>` по секциям (каждая помечена комментарием `BLOCK N`).
- **Интерактив** — в `<script>` перед закрывающим `</body>`.

Изображения встроены как `data:image/...;base64,...`. Чтобы заменить картинку — подготовьте новый base64 и замените соответствующую строку `src`.

## 🔧 Интеграции

- **Meta Pixel** установлен (ID в `<head>`); при отправке формы шлётся событие `Lead`.
- Форма пока **не отправляет данные на бэкенд** — по сабмиту показывается экран благодарности. Для приёма заявок подключите обработчик (CRM, Google‑таблица, Telegram, email‑webhook).
